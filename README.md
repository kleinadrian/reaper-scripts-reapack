# Adrian Klein Scripts

A ReaPack repository for personal scripts used in daily work for sound design, editing, recording, project formatting, and workflow.

## Install in REAPER

In REAPER, go to:

Extensions → ReaPack → Import repositories

Then import this URL:

`https://raw.githubusercontent.com/kleinadrian/reaper-scripts-reapack/master/index.xml`

## Categories

- Automation
- Editing
- Organization
- Recording
- Workflow
